1) display:inline Takes only the space required by the element. No line breaks before and after. Setting width/height (or margin(top/bottm)/padding(top)) not allowed. 2) display: block Takes full available width of it's parent element and leaves a newline before and after the element 3) display: inline-block Similar to inline but setting height, width, margin, and padding is allowed.
